Curso esencial de HTML y CSS

Este es el inicio de tu carrera como Desarrollador Web, empezarás aprendiendo HTML y CSS para la creación de contenidos, en el nivel 2 aprenderás de componentes y en 3 sobre layouts

Setup

1. Google Chrome

Chrome va a ser el navegador que utilizaremos en el curso por su extenso soporte de nuevos estandares y herramientas de desarrollo avanzadas y de amigable uso

2. Visual Studio

Visual Studio será el editor de texto popular y potente del mercado actualmente, su fácil uso y capacidad de extención por medio de plugins la hace completamente perfecto para cualquier programador sin importar el lenguaje en el que se desemvuelva

3. Figma

Figma es la herramienta más popular en la actualidad para entregar diseños a los desarrolladores para su posterior implementación. Por su facíl uso porque puede ser usadi directamente desde el navegador.

4. GitHub

Así como en Facebook te encuentran tus amigos, en GitHub te encuentran otros programadores de todo el mundo, en vez de subir fotos de tus viajes aqui subiras tu código y puedes elegir buscando "open source" o crearlo demanera privada, GiHub será esencial para almacenar tu futuro portafolio
